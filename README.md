# Ahmed Tariq

Cloud DevOps Engineer focused on Azure, AWS, GCP, Terraform, Kubernetes, CI/CD, observability, cloud security, and AI infrastructure.

I design, automate, and operate production-ready cloud platforms for startups and engineering teams.

## Focus Areas

- Cloud infrastructure automation with Terraform, Bicep, ARM Templates, CloudFormation, and AWS CDK
- CI/CD pipelines using Azure DevOps, GitHub Actions, Jenkins, and cloud-native deployment tools
- Containerized deployments using Docker, Kubernetes, AKS, EKS, ECS, and Azure Container Apps
- Monitoring and observability using Azure Monitor, Application Insights, CloudWatch, Prometheus, Grafana, and ELK
- Cloud security, governance, private networking, IAM, Microsoft Defender for Cloud, and AWS security services
- AI infrastructure and production deployment for LLM, chatbot, and automation systems
